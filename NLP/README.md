## 简介
该笔记的内容包含了常见的自然语言处理基本技术与原理，主要包括6个单元：
1. 基础知识
2. 语言模型与朴素贝叶斯
3. 主题模型
4. 隐马尔科夫模型
5. 深度学习
6. Word2Vec
